### Git Ignore
Git Ignore is used when you want git to ignore changes from some files that include in the same directory.
First step: Create a .gitignore file
Second: Open the .gitignore file, adding the file name including extension of files you want to ignore
3rd:  Add and commit the .gitignore file/
4th: Clear cached on your local computer. git rm -r --cached .
After this, any changes on listed files will be ignored when call git status.
